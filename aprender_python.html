Recursos para aprender Python (resumo)

Abaixo estão links úteis com uma breve descrição para quem está começando em Python.

Documentação oficial — Tutorial

docs.python.org — Tutorial oficial do Python. Excelente para entender a linguagem desde os conceitos básicos até tópicos intermediários, com exemplos e orientação formal.

Real Python

realpython.com — Tutoriais práticos e artigos bem explicados sobre Python, cobrindo desde fundamentos até boas práticas e bibliotecas populares.

Programiz — Tutorial Python

programiz.com — Guia passo a passo com exemplos simples, ideal para quem prefere aprender com exercícios curtos e diretos.

Curso em Vídeo (Português)

cursoemvideo.com — Curso em vídeo gratuito em Português (prof. Gustavo Guanabara), muito indicado para iniciantes que preferem aulas gravadas.

Apostilas e materiais brasileiros

  • Apostila PET ADS — São Carlos — material didático em PDF com exercícios.
  • Introdução a Python — IFRN — outro recurso em PDF para estudo estruturado.
  • Introdução à Programação com Python — Nilo Ney — foco em algoritmos e lógica aplicados em Python.

Dica: combine leitura com prática — abra seu editor e execute os exemplos.
