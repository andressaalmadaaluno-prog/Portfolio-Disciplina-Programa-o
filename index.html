Algoritmos e lógica de programação

Um algoritmo é uma sequência de passos finitos para resolver um problema. As apostilas mostram que ele deve ser:

  • claro e objetivo;
  • ordenado;
  • detalhado o suficiente para ser implementado;
  • sem ambiguidades.

Resumo dos conceitos

  • Pseudocódigo facilita a estruturação antes da programação.
  • Fluxogramas ajudam a visualizar decisões e repetições.
  • Variáveis e operações aritméticas são essenciais em problemas de engenharia.
  • Estruturas condicionais e de repetição aparecem em quase todos os exercícios.

Questões para praticar

1. Área de um círculo

Calcule a área de um círculo cujo raio é 0.15 m. Escreva o código em Python.

Resposta (mostrar) 
import math
raio = 0.15
area = math.pi * (raio ** 2)
print(f"Área do círculo: {area:.4f} m²")

2. Conversão de tempo

Converta um total de 3670 segundos em horas, minutos e segundos.

Resposta (mostrar) 
total = 3670
horas = total // 3600
resto = total % 3600
minutos = resto // 60
segundos = resto % 60
print(horas, 'h', minutos, 'm', segundos, 's')

3. Pseudocódigo

Escreva um pseudocódigo que leia três números e mostre o maior entre eles.

Resposta (mostrar) 
ler a, b, c
maior = a
se b > maior então
  maior = b
fimse
se c > maior então
  maior = c
fimse
imprimir maior

4. Trace de programa

Qual será a saída do código Python abaixo?

a = 5
b = 2
c = a // b
print(c)
Resposta (mostrar)

Saída: 2 (divisão inteira)

5. Problema de engenharia simples

Dado tensão = 220 V e resistência = 17.6 Ω, calcule a corrente (I) usando a Lei de Ohm.

Resposta (mostrar) 
tensao = 220.0
resistencia = 17.6
corrente = tensao / resistencia
print(f"Corrente: {corrente:.2f} A")
